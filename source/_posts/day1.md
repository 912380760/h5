## css 的几种形式
* 内联CSS
* 内部css
* 外部文件css

```
// 内联CSS
<div style="color: red;">test</div>

// 内部CSS一般写在head标签里
<style>
.test {
    color: red;
} 
</style>

// 外部CSS，单独的css文件
<link rel="stylesheet" href="../index.css">
```
